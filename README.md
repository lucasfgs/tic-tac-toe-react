# **Tic Tac Toe**

This project was made with [React.js](https://github.com/facebook/react) for learning porpuses

## **Install**

**Install all dependencies**

_With npm:_

```
npm install
```

_With yarn:_

```
yarn install
```

## **Usage**

**Start the app:**

_With npm:_

```
npm start
```

_With yarn:_

```
yarn start
```
